## Key Features

1. **Responsive Web Design**
   - 미디어 쿼리(`@media`)를 활용하여 PC, 태블릿, 모바일에 최적화된 레이아웃을 구현했습니다.
   - 모바일 환경에서는 햄버거 메뉴를 제공하여 사용자 편의성을 높였습니다.

2. **Interactive UI**
   - JavaScript를 이용한 **자격증 슬라이더** 기능 구현.
   - 스크롤에 따른 헤더 고정(`sticky`) 및 호버 애니메이션 적용.

3. **Project Showcasing**
   - 외부 뉴스 API를 연동하고 AI 모델을 활용해 복잡한 정보를 요약하는 시스템 구축.
   - JSON 데이터를 활용한 동적 콘텐츠 로딩 경험(창작 동화 서점 프로젝트).
   - 드래그 앤 드롭 기능을 포함한 인터랙티브 도구 제작 경험(Page Builder Pro).

## Project Structure

- `index.html` / `Mypage.html`: 메인 페이지
- `about.html`: 자기소개, 성격의 장단점, 보유 기술 및 자격증
- `portfolio.html`: 프로젝트 리스트 및 상세 설명
- `contact.html`: 연락처 및 SNS 링크
- `css/`: 디자인 스타일시트 폴더
- `js/`: 인터랙티브 기능 자바스크립트 폴더
- `img/`: 프로젝트 및 프로필 이미지 리소스 폴더
